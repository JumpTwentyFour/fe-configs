# `@jump24/eslint-config-vue-ts`

> ESLint config for Vue 3 with TypeScript by Jump24

## Usage

Install the config:
```bash
yarn add -D @jump24/eslint-config-vue-ts
```

```bash
npm install --save-dev @jump24/eslint-config-vue-ts
```

Add the config:
```js
// .eslintrc.js
module.exports = {
  extends: [
    '@jump24/eslint-config-vue-ts',
  ],
};
```
