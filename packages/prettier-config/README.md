# `@jump24/prettier-config`
@jump24 Prettier config

## Usage

Install the config:
```bash
yarn -D @jump24/prettier-config
```

```bash
npm install --save-dev @jump24/prettier-config
```

Add the config:
```js
// package.json
{
  "name": "my-cool-library",
  "version": "9000.0.1",
  "prettier": "@jump24/prettier-config"
}
```
