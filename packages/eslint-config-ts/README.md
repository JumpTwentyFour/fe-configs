# `@jump24/eslint-config-ts`
@jump24 ESLint config for typpescript browser-based projects

## Usage

Install the config:
```bash
yarn add -D @jump24/eslint-config-ts
```

```bash
npm install --save-dev @jump24/eslint-config-ts
```

Add the config:
```js
// .eslintrc.js
module.exports = {
  extends: [
    '@jump24/eslint-config-ts',
  ],
};
```
