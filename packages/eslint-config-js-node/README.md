# `@jump24/eslint-config-js-node`
@jump24 ESLint config for javascript node-based projects

## Usage

Install the config:
```bash
yarn -D @jump24/eslint-config-js-node
```

```bash
npm install --save-dev @jump24/eslint-config-js-node
```

Add the config:
```js
// .eslintrc.js
module.exports = {
  extends: [
    '@jump24/eslint-config-js-node',
  ],
};
```
