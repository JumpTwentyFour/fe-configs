# `@jump24/eslint-config-js`
@jump24 ESLint config for javascript browser-based projects

## Usage

Install the config:
```bash
yarn add -D @jump24/eslint-config-js
```

```bash
npm install --save-dev @jump24/eslint-config-js
```

Add the config:
```js
// .eslintrc.js
module.exports = {
  extends: [
    '@jump24/eslint-config-js',
  ],
};
```
