# `@jump24/eslint-config-{CHANGE}`
@jump24 ESLint config for {CHANGE} projects

## Usage

Install the config:
```bash
yarn add -D @jump24/eslint-config-{CHANGE}
```

```bash
npm install --save-dev @jump24/eslint-config-{CHANGE}
```

Add the config:
```js
// .eslintrc.js
module.exports = {
  extends: [
    '@jump24/eslint-config-{CHANGE}',
  ],
};
```
